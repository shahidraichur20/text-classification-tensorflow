# TensorFlow Internship Projects

This repository contains three TensorFlow-based Machine Learning and Deep Learning projects completed as part of an internship program.

---

# Project 1: Text Classification using TensorFlow

## Objective

Classify SMS messages as Spam or Ham using TensorFlow and Keras.

## Dataset

SMS Spam Collection Dataset (Kaggle)

## Technologies Used

* Python
* TensorFlow
* Keras
* Pandas
* NumPy
* Matplotlib

## Workflow

1. Load SMS Spam Dataset
2. Preprocess and clean text data
3. Encode labels
4. Split data into training and testing sets
5. Convert text into numerical representations
6. Build and train a neural network
7. Evaluate model performance

## Results

* Test Accuracy: **97.58%**

## File

* `Text_Classification_Using_TensorFlow.ipynb`

# Project 2: Object Detection using TensorFlow

## Objective

Detect and classify handwritten digits while simultaneously predicting their bounding box coordinates.

## Dataset

MNIST Handwritten Digits Dataset

## Technologies Used

* Python
* TensorFlow
* NumPy
* Matplotlib
* Pillow

## Workflow

1. Load MNIST Dataset
2. Generate custom object detection images
3. Create bounding box annotations
4. Build CNN feature extractor
5. Train classification head
6. Train bounding box regression head
7. Evaluate performance

## Results

* Training Accuracy: **99.0%**
* Validation Accuracy: **97.0%**
* Bounding Box Localization using CNN Regression

## File

* `Object_Detection_Using_TensorFlow.ipynb`

# Project 3: Landmark Detection using TensorFlow

## Objective

Classify landmark and scene images using Transfer Learning with VGG19.

## Dataset

Intel Image Classification Dataset (Kaggle)

## Technologies Used

* Python
* TensorFlow
* Keras
* NumPy
* Matplotlib
* VGG19 Transfer Learning

## Workflow

1. Load Intel Image Classification Dataset
2. Preprocess image data
3. Create training, validation, and testing generators
4. Load pre-trained VGG19 model
5. Build transfer learning classifier
6. Train and evaluate the model

## Results

* Training Accuracy: **82.40%**
* Validation Accuracy: **89.87%**
* Test Accuracy: **90.53%**

## File

* `Landmark_Detection_Using_TensorFlow.ipynb`
