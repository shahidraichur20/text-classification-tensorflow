# TensorFlow Internship Projects

This repository contains TensorFlow projects completed as part of an internship program, covering Natural Language Processing (NLP) and Computer Vision concepts.

## Project 1: Text Classification using TensorFlow

### Objective

Classify SMS messages as **Spam** or **Ham (Not Spam)** using TensorFlow and Keras.

### Dataset

SMS Spam Collection Dataset (Kaggle)

### Technologies Used

* Python
* TensorFlow
* Keras
* Pandas
* NumPy
* Matplotlib

### Workflow

1. Load SMS Spam Dataset
2. Preprocess and clean text data
3. Encode labels
4. Split data into training and testing sets
5. Vectorize text data
6. Build and train a neural network
7. Evaluate model performance
8. Visualize results

### Results

* Test Accuracy: **97.58%**

### File

* `Text_Classification_Using_TensorFlow.ipynb`

## Project 2: Object Detection using TensorFlow

### Objective

Detect and classify handwritten digits while simultaneously predicting their bounding box coordinates.

### Dataset

MNIST Handwritten Digits Dataset

### Technologies Used

* Python
* TensorFlow
* NumPy
* Matplotlib
* Pillow

### Workflow

1. Load MNIST Dataset
2. Create custom object detection dataset
3. Generate random bounding box locations
4. Build CNN feature extractor
5. Train classification head
6. Train bounding box regression head
7. Evaluate model performance
8. Visualize predictions and localization results

### Results

* Training Accuracy: **99.0%**
* Validation Accuracy: **97.0%**
* Bounding Box Localization using CNN Regression

### File

* `Object_Detection_Using_TensorFlow.ipynb`
